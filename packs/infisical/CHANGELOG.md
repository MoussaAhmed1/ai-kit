# Changelog

All notable changes to the infisical plugin will be documented in this file.

## [0.1.0] - 2026-02-18

### Added
- Initial release (experimental)
- 1 agent: infisical-ops (secret management operations specialist)
- 5 commands: infisical-init, infisical-auth, infisical-scan, infisical-env-sync, infisical-ci-setup
- 3 skills: infisical-patterns, infisical-ci-integration, secret-hygiene
- Support for cloud and self-hosted Infisical instances
- CI/CD integration for GitHub Actions, GitLab CI, CircleCI, Jenkins, Bitbucket
- Docker, Kubernetes, and serverless deployment patterns
